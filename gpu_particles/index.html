GPU Particles

Particle simulations - this time in parallel! Uses textures to store and update positions and velocity information. This way the simulations can be run on the GPU as a simple pixel shader. Since the GPU can't write to the same texture being read, two textures have to be used and swapped each frame.

I'll hopefully be able to reuse this as a template for upcoming projects.

Due to WebGL issues, the simulation might not work well on your device
